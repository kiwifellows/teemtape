Comment on stocks, together.

An open-source ticker app for leaving anonymous notes next to stock symbols — from the web, mobile, or the CLI. It is a tool for learning how to build software incrementally with agentic AI. Not a trading app.

Delayed ~1 min Anonymous notes Shareable links Cloudflare Workers + D1
Phase 1 — Wireframes for approval. These are clickable HTML mockups (no backend). Open each one, try the note popups and share link, toggle light / dark mode (top right), and approve the layout/flows before we start building the real app.

Features in scope

  • ✓ Table list of stocks
  • ✓ Backend connects to Polygon/Massive free API
  • ✓ Prices delayed ~1 minute (not real-time)
  • ✓ Anonymous notes on each symbol
  • ✓ Notes from agents (CLI) and users (web)
  • ✓ Note popup when viewing a symbol
  • ✓ Personal link with MD5 token in the URL
  • ✓ Shareable with friends / family / traders

How it fits together

CLI → Web / Mobile (React) → Cloudflare Worker API → D1 (notes) + Polygon/Massive (delayed quotes)

See docs/architecture.md for the full plan, including the wrangler.toml setup and how the kiwifellows practices guides are applied.
